# InsisVim

An out-of-the-box Neovim IDE layer that setup development environment in an incredibly simple way.

```lua
require("insis").setup({
  golang = {
    enable = true,
    lsp = "gopls",
    linter = "golangci-lint",
    formatter = "gofmt",
    format_on_save = true,
  },
})
```
